Opções de Salvamento

  • Formato: permite a escolha do formato de salvamento. Dependendo dos recursos do formato, você pode escolher se deseja salvar o arquivo comprimido em vários volumes e se deseja usar criptografia por senha;
  • Compressão: você pode escolher a intensidade de compressão para a maioria dos formatos. Uma compressão melhor sempre resulta em uma operação mais lenta;
  • Criptografia: alguns formatos (7-ZIP, ZIP, RAR) podem ser criptografados. Use uma criptografia ZIP fraca somente se absolutamente necessário, pois os dados não ficarão realmente seguros. Na maioria das vezes, é preferível usar a criptografia Forte (AES-256) — também conhecida como criptografia de nível militar, o que significa que não há maneira conhecida de burlar esse tipo de criptografia atualmente. Se você escolher a criptografia Forte (AES-256) com criptografia de cabeçalho (disponível apenas em 7-Zip e RAR), até os nomes dos itens serão criptografados e não poderão ser listados sem que a senha correta seja digitada. Não perca ou esqueça as suas senhas, pois ninguém poderá ajudar você. Nota: o Utilitário de Compressão do macOS é compatível apenas com a criptografia ZIP fraca, mas a maioria dos utilitários de arquivos comprimidos de outros desenvolvedores são compatíveis com todos os tipos;
  • Senha: clique em Definir para digitar uma senha. Se você deixar esse campo em branco, o BetterZip solicitará uma senha quando necessário;
  • Dividir em volumes de: para dividir o arquivo comprimido em vários volumes, digite o tamanho desejado no campo de texto e escolha uma unidade (KB, MB, GB) no menu local ao lado. Caso não queira dividir o arquivo comprimido, deixe o campo em branco;
  • Parâmetros adicionais: alguns formatos podem ter mais opções. Você pode digitá-las nesse campo, mas certifique-se de saber o que faz;
  • Comentários: os formatos ZIP e RAR permitem a adição de comentários ao arquivo comprimido. Adicione o comentário diretamente no campo de texto ou clique no botão Editar ao lado do comentário para ver um editor de comentários maior;
  • Executar script ao concluir: depois de criar o arquivo comprimido, o BetterZip executará o script de shell ou comando digitado, passando o nome do arquivo comprimido;
  • Remover itens invisíveis dos arquivos comprimidos: o BetterZip também pode remover itens invisíveis do Mac do arquivo comprimido. Um ZIP criado em um Mac normalmente contém vários itens ocultos que apenas outros Macs entendem, como ajustes do Finder e ramificações de recurso. Para usuários de Windows ou Linux, esses itens são completamente inúteis;
  • Excluir padrões: defina padrões para fazer a correspondência de itens que não devem ser incluídos no arquivo comprimido;
  • Padrão é uma expressão regular: selecione para usar a correspondência por expressão regular ou desmarque para usar a correspondência por curingas * e ?;
  • Definir data do arquivo comprimido pelo item mais recente: ao criar um arquivo comprimido de um item datado de 8 de novembro de 2004, normalmente o arquivo comprimido usará a data de hoje. Ative essa opção para que ele herde a data original do item mais recente;
  • Fechar janela ao concluir corretamente;
  • Criar arquivos comprimidos individuais ao usar fila de operações: se você usar esse pré-ajuste para comprimir vários itens através da fila de operações, cada item principal será salvo em um arquivo comprimido individual. As operações em fila incluem: soltar itens na Barra de Operações, serviços, ações do Automator, botão do BetterZip em janelas do Finder e qualquer outra operação que passe pela janela da fila de operações;
  • Não incluir pasta raiz única no arquivo comprimido: se apenas uma pasta for incluída em um arquivo comprimido, a pasta pode ser omitida. Isso faz com que apenas o conteúdo da pasta seja incluído no arquivo comprimido. Essa opção também funciona com a opção de arquivos comprimidos individuais, descrita anteriormente. Por exemplo, você pode comprimir cinco pastas e obter cinco arquivos comprimidos apenas com os conteúdos das pastas, sem as pastas em si.
