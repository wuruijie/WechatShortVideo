Salve um Arquivo Comprimido

Para salvar um arquivo comprimido, escolha Arquivo > Salvar. No painel de salvamento, escolha a pasta de destino e especifique as opções de salvamento. Clique então em Salvar.

Para salvar um arquivo comprimido usando um dos pré-ajustes configurados, escolha Arquivo > Salvar com Pré-Ajuste ou mantenha o botão Salvar pressionado na barra de ferramentas e escolha o pré-ajuste no menu local exibido após um momento.
Você pode clicar rapidamente em Salvar na barra de ferramentas para criar um arquivo comprimido usando o ajuste padrão. Se nenhum pré-ajuste padrão de salvamento estiver definido, esse clique abrirá o painel de salvamento, onde você poderá definir as opções desejadas. Caso mantenha a tecla ⌥ pressionada ao escolher um pré-ajuste no menu da barra de ferramentas, o BetterZip exibirá o painel de salvamento padrão, onde você poderá substituir as opções do pré-ajuste.
