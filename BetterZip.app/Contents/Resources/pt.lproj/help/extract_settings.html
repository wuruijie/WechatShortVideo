Opções de Extração

  • Excluir padrões: defina padrões para fazer a correspondência de itens que não devem ser extraídos do arquivo comprimido;
  • Padrão é uma expressão regular: selecione para usar a correspondência por expressão regular ou desmarque para usar a correspondência por curingas * e ?;
  • Criar pasta extra;
  • Mostrar itens extraídos no Finder;
  • Abrir itens extraídos;
  • Fechar janela do arquivo comprimido;
  • Mover arquivo comprimido para o Lixo após extração bem-sucedida;
  • Executar script ao concluir: depois de extrair um arquivo comprimido, o BetterZip executará esse script de shell ou comando, passando o nome do arquivo comprimido como primeiro parâmetro e o destino da extração como segundo;
  • Substituir arquivos no destino por arquivos extraídos sem avisar: use esse ajuste com cuidado e não o use caso não costume fazer backups regularmente.
