Adicione Itens a um Arquivo Comprimido

Clique no botão Adicionar na barra de ferramentas ou escolha Arquivo > Adicionar. O painel de abertura será exibido, onde você poderá selecionar um ou vários itens para adicionar ao arquivo comprimido. Para adicionar os novos itens a uma pasta existente dentro do arquivo comprimido, selecione a pasta antes de usar o comando Adicionar.

Você também pode arrastar itens do Finder para a janela do arquivo comprimido. Se você soltar os novos itens dentro de uma pasta, eles serão adicionados à ela. As pastas abrem-se automaticamente, assim como no Finder.
