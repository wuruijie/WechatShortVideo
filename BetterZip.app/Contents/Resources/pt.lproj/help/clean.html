Filtre Itens e Limpe Arquivos Comprimidos

A opção “Apagar com Padrão” permite a remoção de itens do arquivo comprimido que coincidam com um padrão de filtragem específico. Por exemplo, se você usa um sistema de controle de versões como o Subversion, é provável que você não deseje incluir pastas ocultas “.svn” nos arquivos comprimidos. O BetterZip usa os padrões de filtragem (simples padrões de correspondência de arquivo que usam curingas * e ? ou expressões regulares) para buscar esses itens, removendo-os com um simples clique. Da mesma forma, o BetterZip pode filtrar itens extraídos de arquivos comprimidos, como arquivos “Thumbs.db” usados pelo Windows para armazenar miniaturas.

Para gerenciar os padrões de busca, use o painel Filtros das preferências.

Para apagar itens com um padrão de filtragem, mantenha a tecla ⌥ pressionada e clique em Apagar na barra de ferramentas ou escolha “Apagar com Padrão” no menu Editar. Você pode usar expressões regulares ou padrões de correspondência de arquivo. É possível usar vários padrões, concatenando-os com ponto e vírgula, como “*.o;*.log”. A correspondência entre cada caminho de item e cada padrão será efetuada.

Para filtrar itens automaticamente ao extrair ou para excluir a possibilidade de adição de itens ao arquivo comprimido, você pode definir padrões de filtragem para cada pré-ajuste.

Para saber o que são expressões regulares e como elas funcionam, consulte este artigo da Wikipédia ou busque tutoriais no DuckDuckGo.
