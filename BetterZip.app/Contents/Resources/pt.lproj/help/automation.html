Integre o BetterZip 4 a ferramentas de produtividade

Ações do Automator

O BetterZip adiciona duas ações ao Automator:

  • Comprimir com BetterZip

    Essa ação pode ser configurada ao selecionar um dos pré-ajustes de salvamento atuais. Ela requer itens como entrada e gera, como saída, arquivos comprimidos.

  • Extrair com BetterZip

    Essa ação pode ser configurada ao selecionar um dos pré-ajustes de extração atuais. Ela requer arquivos comprimidos como entrada e gera, como saída, os itens contidos no arquivo comprimido.

Serviços

Em versões anteriores, o BetterZip disponibilizava dois serviços para outros apps (e, obviamente, para o Finder) através do menu Serviços. O BetterZip 4 permite que você adicione quantos serviços desejar ao menu de Serviços. Acesse o painel Pré-Ajustes das preferências do BetterZip, escolha o pré-ajuste que deseja adicionar como serviço, clique no botão de engrenagem abaixo da lista de pré-ajustes, escolha “Adicionar este Pré-Ajuste ao menu de Serviços”, digite um nome para o serviço e clique em OK.

Para usar um serviço, selecione itens no Finder e escolha o serviço no menu Finder > Serviços (ou no submenu Serviços do menu contextual).

Você pode até definir atalhos de teclado para os serviços do BetterZip, como ^⌥⌘C para comprimir e ^⌥⌘E para extrair. Para definir os atalhos dos serviços, abra as Preferências do Sistema, clique em Teclado, selecione a aba Atalhos e, na lista à esquerda, selecione Serviços. Na lista à direita, role até “Arquivos e Pastas”, escolha Comprimir com BetterZip e clique em Adicionar Atalho. Um campo de texto editável é exibido. Digite o atalho desejado.

Para facilitar a remoção de serviços, o menu do botão de engrenagem também oferece a opção “Mostrar Pasta de Serviços no Finder”. Para remover um serviço, basta apagá‑lo dessa pasta.

O BetterZip é compatível com alguns dos utilitários de produtividade mais populares:

Fluxos de Trabalho do Alfred

Caso não tenha instalado os fluxos de trabalho do Alfred ao abrir o BetterZip pela primeira vez, selecione o menu BetterZip > Instalar Fluxos de Trabalho do Alfred para instalá‑los e adicionar duas ações de arquivos ao Alfred: “Extrair com BetterZip” (para arquivos comprimidos) e “Comprimir com BetterZip” (para outros itens). Escolha uma das ações do BetterZip para ver uma lista de pré-ajustes. Escolha um dos pré-ajustes para que o BetterZip comprima ou extraia os itens.

Ações do LaunchBar

O LaunchBar é uma outra ferramenta de produtividade popular. As ações do BetterZip para o LaunchBar se assemelham mais a serviços do que os fluxos de trabalho do Alfred, já que são pré-configuradas com um pré-ajuste específico. Acesse o painel Pré-Ajustes das preferências do BetterZip, escolha o pré-ajuste que deseja adicionar como Ação do LaunchBar, clique no botão de engrenagem abaixo da lista de pré-ajustes, escolha “Adicionar este Pré-Ajuste ao LaunchBar 6”, digite um nome para o serviço e clique em OK.

Ações do Dropzone

A criação de ações do Dropzone funciona da mesma forma que a do LaunchBar. Acesse o painel Pré-Ajustes das preferências do BetterZip, escolha o pré-ajuste que deseja adicionar como Ação do Dropzone, clique no botão de engrenagem abaixo da lista de pré-ajustes, escolha “Adicionar este Pré-Ajuste ao Dropzone 3”, digite um nome para o serviço e clique em OK.

Integração com o Hazel

O Hazel é uma central de controle poderosa para várias tarefas de automação de arquivos e funciona perfeitamente com o BetterZip 4. A interação entre eles funciona melhor com AppleScripts. Embora o Hazel possua funções para criar arquivos comprimidos, o BetterZip pode ajudar caso você precise de formatos além de Zip ou queira usar criptografia.

Este é um script simples que pode ser adicionado como ação ao Hazel para comprimir itens com o BetterZip: 

tell application "BetterZip"
	archive theFile with preset "ZipArrastado"
end tell

theFile é o arquivo fornecido pelo Hazel aos scripts, enquanto ZipArrastado é um pré-ajuste configurado no BetterZip. Caso prefira, você pode incluir todas as opções do BetterZip no script em vez de usar um pré-ajuste.

Controle do BetterZip via AppleScript

A compatibilidade com a linguagem AppleScript é a base de toda a automação do BetterZip. Para consultar a definição completa dos scripts, abra o Editor de Scripts, escolha Arquivo > Abrir Dicionário, selecione o BetterZip na lista e clique em Escolher.

Controle do BetterZip via Python

Você também pode usar Python (e outras linguagens) e criar scripts para o BetterZip através de Scripting Bridge. Mais adiante, você verá um exemplo rápido onde a matriz de nome de arquivo items é comprimida. Ao realizar um queuedArchive ou queuedUnarchive, um id que retorna a porcentagem do progresso (0–100) e pode ser usado em calls subsequentes é fornecido. O call de saída retorna os arquivos comprimidos produzidos. Pelo BetterZip ser encerrado dois segundos após o término das operações de script, não use um intervalo de pesquisa longo (principalmente no final).


import ScriptingBridge
import time

preset = 'Zip Limpo'

BetterZip = ScriptingBridge.SBApplication.applicationWithBundleIdentifier_("com.macitbetter.betterzip")

pid = BetterZip.queuedArchive_withPreset_withOptions_(items, preset, {})
if pid > 0:
	a = BetterZip.progress_(pid)
	while a < 100 and BetterZip.isRunning():
		a = BetterZip.progress_(pid)
		# a is the percentage done; do something with it?
		time.sleep (1)
	archives = BetterZip.output_(pid)
else:
	# handle error
