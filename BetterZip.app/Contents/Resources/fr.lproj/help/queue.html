La file d'attente d'opération

BetterZip 2 dispose d'une file d'attente pour l'exécution de multiples opérations sur des archives dans un ordre prédéterminé. Vous pouvez laisser choisir à macOS le nombre d'opérations simultanées autorisées ou le définir manuellement.

La file d'attente sera utilisée automatiquement quand vous démarrez une opération sur une archive en utilisant l'un des services de BetterZip, quand vous réalisez une extraction directe (en maintenant enfoncées les touches ^⌘ pendant que vous double-cliquez sur une archive dans le Finder), ou quand vous déposez des fichiers ou dossiers dans la fenêtre de file d'attente.
