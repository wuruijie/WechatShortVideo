Le gestionnaire de mots de passe

BetterZip possède une nouvelle fonction pratique : le gestionnaire de mots de passe d'archive. BetterZip peut enregistrer tous les mots de passe que vous saisissez en ouvrant une archive et les sauvegarder dans votre trousseau macOS. À chaque fois que vous ouvrez une archive cryptée, BetterZip peut essayer les mots de passe qu'il contient et utiliser automatiquement celui qui convient. BetterZip peut aussi afficher tous les mots de passe dans une liste contenue dans le volet des mots de passe, de manière à vous permettre de choisir facilement celui qu'il faut utiliser. Pour des raisons de sécurité, cette fonctionnalité peut être totalement désactivée.

Pour activer le gestionnaire de mots de passe et modifier la liste des éléments qu'il contient, allez dans l'onglet Mots de passe dans la fenêtre des préférences. Vous devrez définir à la première utilisation un mot de passe principal qui protège les mots de passe et les réglages du gestionnaire. Pour cela, cliquez sur le bouton Définir le mot de passe principal, et entrez deux fois le mot de passe souhaité. Désormais, vous pouvez cliquer sur le cadenas, puis entrer le mot de passe principal, pour déverrouiller l'accès aux paramètres du gestionnaire de mots de passe.

Le menu en haut vous permet de choisir comment les mots de passe de la liste doivent être utilisés. Vous noterez la présence d'une entrée Tester automatiquement tous les mots de passe, excepté pour les archives RAR. Avec certaines archives RAR, il n'est possible de savoir si le mot de passe utilisé est valide ou non qu'après l'extraction complète de l'archive. Ainsi, il se peut que BetterZip ait à extraire entièrement l'archive plusieurs dizaines de fois avant d'essayer le bon mot de passe. Il est donc préférable d'éviter un tel comportement en excluant les archives RAR. Sauf si vous savez ce que vous faites.
