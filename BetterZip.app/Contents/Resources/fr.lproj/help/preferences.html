Changer l'apparence de BetterZip

Mode d'affichage

BetterZip a deux manières d'afficher le contenu d'une archive :

  • Liste classique (⌘1)
    Tous les fichiers et dossiers sont affichés sous forme d'une liste traditionnelle ne tenant pas compte de la hiérarchie de l'archive. Si vous utilisez ce mode, vous devriez afficher la colonne Chemin. Ce mode est particulièrement utile si vous avez besoin de trier des fichiers sans vous inquiéter de leur localisation dans la hiérarchie de l'archive (pour effectuer un tri par date de modification, par exemple).
  • Arborescence (⌘2)
    Les fichiers sont affichés dans un arbre, rangés dans leurs dossiers respectifs, comme la vue Colonne du Finder.

Taille de la police

Vous pouvez augmenter la taille de la police utilisée dans la fenêtre principale en appuyant sur ⌘+ et vous la diminuer en appuyant sur ⌘-.

Personnaliser BetterZip — Les Préférences

Onglet Général

Démarrer avec une archive vierge
Activez cette option si vous souhaitez que BetterZip s'ouvre sur une archive vide à chaque démarrage. Lorsque vous ouvrez une archive existante, les fenêtres d'archive vide dont il est question sont réutilisées pour en afficher le contenu.

Quitter après la fermeture de la dernière fenêtre ouverte
Activez cette option si vous souhaitez que BetterZip soit automatiquement quitté dès que toutes les fenêtres ont été fermées.

Décompresser automatiquement à l'ouverture
Au lieu d'afficher le contenu d'une archive ouverte depuis le Finder, BetterZip peut la décompresser immédiatement. Ce procédé est appelé Extraction directe. Activez cette option si vous souhaitez activer l'extraction directe. Vous pouvez ponctuellement activer/désactiver cette fonction en maintenant les touches ^⌘ (ctrl+cmd) enfoncées lors de l'ouverture d'une archive.

Réinitialiser les messages "Ne plus afficher"
Appuyez sur ce bouton pour que BetterZip affiche à nouveau tous les messages d'alertes que vous aviez précédemment décidé d'ignorer définitivement en cochant la case "Ne plus afficher".

Onglet Modules

Pour le moment, BetterZip ne peut utiliser qu'un module externe qui est rar et qui vous permet de créer des archives rar. L'extraction est intégrée directement à BetterZip, vous n'avez donc pas besoin d'un module externe pour cela.

Onglet Nettoyage

La fonction Nettoyage vous permet de supprimer certains fichiers automatiquement d'une archive. Si vous utilisez un système de gestion de versions tel que CVS ou Subversion, vous ne souhaitez probablement pas inclure dans vos archives les dossiers cachés CVS et .svn. BetterZip utilise des modèles prédéfinis de noms de fichiers (expressions régulières, ou rationnelles) pour trouver de tels fichiers et les éliminer d'un seul clic. Dans cet onglet, vous pouvez ajouter de nouveaux modèles de noms de fichiers (ou expressions régulières) en cliquant sur le bouton + sous la liste ou en enlever en cliquant sur le bouton -. Cliquez sur un des modèles pour l'éditer.

BetterZip peut filtrer les fichiers à deux occasions : avant l'enregistrement ou après l'extraction d'une archive. Utilisez les cases à cocher dans la liste pour définir les moments où chaque modèle de nom de fichier doit être utilisé.

Pour en savoir plus sur les expressions rationnelles et apprendre à les utiliser, consultez cet article sur Wikipédia ou utilisez Google pour trouver de nombreux exemples.

Onglet Types de fichiers

Dans la liste, tous les types de fichiers que BetterZip peut gérer sont affichés. Vous pouvez choisir lesquels vous souhaitez effectivement associer à BetterZip. Quand vous choisissez de définir BetterZip comme application préférée pour un type de fichier, l'application le gérant précédemment est sauvegardée et vous pouvez la restaurer de la même manière que vous avez associé BetterZip à ce type de fichier.

Onglet Mots de passe

Dans cet onglet, vous pouvez gérer les mots de passe que vous utilisez souvent. Consultez la page Le gestionnaire de mots de passe d'archive pour plus d'informations.

Onglet Avancé

Dans cet onglet vous pouvez gérer quelques options complémentaires :

Nombre d'opérations simultanées : Quand vous utilisez la file d'attente d'opérations à effectuer sur une archive, BetterZip peut limiter le nombre d'opérations exécutées en même temps. Si cette valeur est définie sur Automatique (recommandé), macOS décidera du nombre d'opérations simultanées autorisées en fonction des ressources système disponibles.

Dossier temporaire
Vous pouvez définir la localisation du dossier temporaire. Si vous créez fréquemment des archives contenant des fichiers stockés sur des disques autres que celui contenant le système de démarrage, définir le dossier temporaire un autre disque accélérera notablement l'ajout de fichiers aux archives.

Ouvrir les fichiers d'un type inconnu avec
Quand vous souhaitez ouvrir un fichier d'un type inconnu, BetterZip utilisera automatiquement cette application (par défaut, le Finder). Si vous utilisez un logiciel de remplacement pour le Finder - comme Path Finder -, vous pouvez définir cette application ici. Vous pouvez également utiliser un éditeur hexadécimal ou un traitement de texte, par exemple.
