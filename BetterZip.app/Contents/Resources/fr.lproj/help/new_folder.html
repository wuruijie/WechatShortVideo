Créer de nouveaux dossiers

Sélectionnez le fichier dans lequel vous souhaitez créer un nouveau dossier. Si vous ne sélectionnez aucun dossier, le nouveau répertoire sera créé à la racine de l'archive. Cliquez sur le bouton Nouveau dossier de la barre d'outils ou choisissez Fichier > Créer un nouveau dossier ou appuyez sur les touches ⌘⇧N de votre clavier. Vous pouvez alors renommer le dossier nouvellement créé comme vous le feriez dans le Finder.
