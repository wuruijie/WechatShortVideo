Rechercher dans une archive

Saisissez n'importe quelle partie du nom de fichier que vous cherchez dans le champ de recherche de la barre d'outils. Les fichiers et dossiers dans la liste seront filtrés et seules les correspondances exactes seront conservées. Pour annuler la recherche et afficher à nouveau tous les fichiers et dossiers, cliquez sur le petit bouton x dans le champ de recherche. Le système de recherche fonctionne de la même manière que celui d'iTunes ou des autres applications. Le raccourci clavier permettant d'entrer dans ce champ de recherche est ⌘F.

Vous pouvez utiliser toutes les commandes lorsque la liste des éléments est filtrée (c'est-à-dire lorsque le champ de recherche n'est pas vide). Attention : Si vous supprimez un dossier, tous les fichiers et sous-dossiers qu'il contient (qu'ils soient visibles ou non) seront également supprimés.
