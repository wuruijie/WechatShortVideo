Enregistrer une archive

Pour enregistrer une archive, choisissez Fichier > Enregistrer. Dans le navigateur de sauvegarde, choisissez le dossier de destination et spécifiez les réglages de l'archive. Cliquez enfin sur le bouton Enregistrer.

Pour enregistrer une archive en utilisant l'un des préréglages configurés, choisissez Fichier > Enregistrer selon un préréglage > Votre préréglage ou cliquez et maintenez enfoncé le bouton Enregistrer dans la barre d'outils et choisissez le préréglage souhaité dans le menu qui apparaître après un instant.
Vous pouvez créer une archive en utilisant le préréglage par défaut en cliquant une seule fois sur le bouton Enregistrer de la barre d'outils (sans maintenir le clic). Si aucun préréglage par défaut n'a été défini précédemment, ce simple clic appellera le navigateur de sauvegarde traditionnel dans lequel vous pourrez définir toutes les options souhaitées. Si vous maintenez la touche ⌥ enfoncée pendant que vous sélectionnez un préréglage dans le menu associé au bouton Enregistrer de la barre d'outils, BetterZip affichera le navigateur de sauvegarde standard en dupliquant les paramètres du préréglage sélectionné, vous permettant de les écraser.
