Préréglages

Les préréglages rassemblent tous les paramètres qui peuvent être appliqués à une opération sur une archive, du dossier de destination et son format aux mots de passe et même au script à exécuter à la fin de l'opération définie. Il en existe deux sortes : les préréglages de création pour la création d'archives et les préréglages d'extraction.

Vous pouvez configurer les préréglages dans la fenêtre des préférences, dans l'onglet Préréglages. L'onglet Préréglages contient lui-même deux onglets, un pour chaque type : sauvegarde et extraction. Ils partagent la même structure : sur la droite, ils ont chacun une liste de tous les préréglages du type correspondant à celui sélectionné. Sous cette liste se trouvent les boutons habituels pour l'ajout et la suppression de paramètres. Un autre bouton contient un menu avec des commandes permettant de dupliquer le préréglage sélectionné ou d'ajouter à nouveau les préréglages par défaut, dans le cas où vous les auriez effacés.

Sur la droite de la liste, vous trouverez en bas une boîte dans laquelle vous pouvez cocher deux cases, si vous souhaitez utiliser le préréglage sélectionné en tant que préréglage par défaut ou paramètre de service. Le préréglage par défaut est celui utilisé pour la décompression en un clic dans la barre d'outils, sans maintien sur le bouton qui a pour effet d'afficher la liste de tous les autres préréglages d'extraction.
Le préréglage de service sera utilisé lorsque vous extrairez ou compresserez une archive en utilisant le menu Services du Finder, que vous utiliserez les raccourcis clavier configurés dans le Finder, que vous effectuerez une extraction directe (en maintenant enfoncées les touches ^⌘ pendant que vous double-cliquez sur une archive dans le Finder) ou que vous déposez simplement des fichiers dans la fenêtre de liste d'attente.

À droite de la liste, une zone vous permet de définir toutes les options qui existent pour le préréglage :

  • Le nom du préréglage. Ce nom sera affiché dans le menu.
  • Le dossier de destination dans lequel l'archive ou les fichiers extraits seront sauvegardés. Déroulez le menu du bouton pour choisir un dossier différent ou demander à BetterZip d'afficher un navigateur de fichier lors de la sauvegarde ou de l'extraction avec ce préréglage. Il y a également un réglage Identique à l'original qui enregistrera l'archive ou les fichiers extraits dans le même dossier que les fichiers originaux ou que l'archive, respectivement.
