サポートしているアーカイブの形式

  • BetterZip が開いて作成することができるアーカイブの形式: ZIP, TAR, TGZ, TBZ, TXZ, 7-Zip, それから— 外部コマンドラインユーティリティを使用して — RAR。
  • さらに、BetterZip が開いて展開することができるアーカイブの形式: Apple ディスクイメージ, TNEF (winmail.dat), ARJ, LZH, ISO, CHM, CAB, CPIO, DEB, RPM, StuffIt の SIT, BinHex, MacBinary.
  • これらのアーカイブは通常以下の拡張子が付けられています: .zip, .epub, .jar, .war, .ear, .sar, .par, .wsr, .rar, .cbz, .cbr, .tar, .tgz, .tbz, .txz, .gz, .bz, .bz2, .cpio, .cpgz, .7z, .dmg, .lha, .lzh, .arj, .iso, .chm, .cab, .deb, .rpm, .sit, .hqx, .bin.
  • BetterZip はファイルの拡張子が、 001, 002, ... といった順番を表す数になっている分割ファイルを結合することもできます。

To configure which of these file types should be handled by BetterZip by default, e.g., when you double-click a file in Finder, go to BetterZip » Preferences » File Types and select the types to handle.

To add your own file extensions to this list, scroll to the last entry in the list named “User-Defined Archive” and click the field in the column “Extensions”. Then enter the extensions without a dot, separated by commas.
